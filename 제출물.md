# 구현 메모
- AI 도구를 활용해 CSV 구조를 확인하고 정적 페이지(`index.html`)와 Python 자동화 도구(`solution.py`)를 작성했습니다.
- 정적 페이지는 재난유형/발생지역 필터, 결과 건수 표시, 재난 카드, 안전시설 지역별·시설유형별 요약을 포함합니다.
- korea.kr 공식 정책뉴스 RSS(`https://www.korea.kr/rss/policy.xml`)를 최신 5건 표시 대상으로 연결했습니다.
- `solution.py`는 표준 라이브러리만 사용하며, 피해금액과 복구기간 결측치는 각 컬럼 유효값 중앙값으로 대체한 뒤 평균을 산출합니다.
- 현재 데이터 기준 재난 500건, 안전시설 100건을 분석 대상으로 사용했습니다.
